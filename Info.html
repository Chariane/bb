Petite clarification sur le développement Web


Le développement web est l'une des sous-branches de l'informatique.

Il consiste à la création d'un site web en entier tel que l'interphase utilisateur et l'interphase serveur. Il reqiert l'apprentissage de plusieurs langages de programmation.

Ici nous allons beaucoup plus nous intéresser aux langages HTML qui est un langage de balissage et au CSS qui est un langage de style.

Autrement, le langage HTML sert à écrire des pages Web tandisque le langage CSS à donner des Design à ces pages.
